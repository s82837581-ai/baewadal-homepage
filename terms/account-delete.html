모디트(Modit) 계정 및 데이터 삭제 요청 안내

본 페이지는 모디트(Modit) 앱 사용자의 계정 삭제 및 개인정보 삭제를 요청하기 위한 공식 안내 페이지입니다.

1. 삭제 요청 방법

아래 이메일로 계정 또는 데이터 삭제 요청을 보내주세요.

이메일: baewadal@baewadal.co.kr

이메일 제목은 다음과 같이 작성해주세요.

[모디트] 계정/데이터 삭제 요청

본문에는 다음 정보를 포함해주세요:

  • 이름
  • 전화번호 (가입 시 입력 정보)
  • 삭제 요청 범위 (전체 삭제 / 특정 데이터 삭제)

2. 처리 절차

  • 요청 확인 후 최대 7일 이내 처리됩니다.
  • 삭제 완료 시 이메일로 안내드립니다.
  • 법적 보관 의무가 있는 정보는 관련 법령에 따라 일정 기간 보관될 수 있습니다.
